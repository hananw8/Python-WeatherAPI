# Python-WeatherAPI
A simple Python script that fetches live weather data using the OpenWeather API. It displays current temperature, daily high, and daily low for selected cities using latitude and longitude. 
# Features
Fetches live weather data from OpenWeather.
Supports metric and imperial units.
Uses environment variables for API key security.
Easy to extend with more cities.
# Requirements
- Python 3.x
- requests
- python-dotenv
